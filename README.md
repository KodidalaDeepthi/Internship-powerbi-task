# Power BI Sales Dashboard

📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze and visualize business data. The dashboard provides meaningful insights through various charts, KPIs, and filters, enabling users to make data-driven decisions.

📊 Features
- Interactive dashboard with slicers
- KPI Cards
- Sales trend analysis
- Category-wise performance
- Regional analysis
- Dynamic visualizations
- Easy-to-understand reports

🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

 📈 Dashboard Insights
- Monitors overall business performance.
- Identifies top-performing categories.
- Compares regional sales performance.
- Tracks sales trends over time.
- Highlights important KPIs for decision-making.

 🎯 Skills Demonstrated
- Data Visualization
- Data Cleaning
- Power Query
- DAX
- Dashboard Design

- Data Analysis
